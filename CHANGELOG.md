## Release Notes

### Version Numbering
This project follows Semantic Versioning:
- **Major version** (X.0.0): Breaking changes or major feature additions
- **Minor version** (1.X.0): New features, backward compatible
- **Patch version** (1.0.X): Bug fixes and minor improvements

### Upgrade Guidelines

**From 1.1.x to 1.2.x:**
- No breaking changes
- Settings migrate automatically
- Backup recommended before upgrade

**From 1.0.x to 1.1.x:**
- User preferences preserved
- New features available immediately
- Review updated documentation for new capabilities

### Support Policy
- Latest major version: Full support
- Previous major version: Security updates only
- Older versions: No longer supported

### Getting Updates
- Check for latest versions through application update feature
- Enable automatic update notifications in settings
- Review documentation for information about new releases

## Reporting Issues

Found a bug or have a feature request?
